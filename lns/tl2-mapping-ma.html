THIẾT LẬP MAPPING 4 MÃ – QUY TẮC TẠO MÃ LNS
⚙️ QUY TẮC TẠO MÃ – Kết hợp 4 thành phần:
(1) Mã Tôm + (2) Mã NL + (3) Mã Công đoạn + (4) Mã Size → Mã LNS (Output)
N1  Loại tôm + N.Liệu + C.Đoạn + Size  →  Full mã (đủ 4 thành phần)
N2  Loại tôm = W + N.Liệu + C.Đoạn + Size  →  Mã riêng cho tôm Wct
N3  N.Liệu + C.Đoạn (nếu NL = Thường → bỏ NL)  →  Mã rút gọn (không tôm, không size)
📐 Ví dụ sinh mã tự động:
T + TH + PTO + N1 → LXPTO
W + VR + PTO + N2 → WVXPTO
TH(bỏ) + PTO → LRTPTO
Sửa STT (1) Mã Tôm (2) Loại NL (3) Mã CĐ Tên Công đoạn (4) Size Tiến trình ĐVT Mã nội bộ → Mã LNS Ghi chú Sử dụng
IMPORT THIẾT LẬP MAPPING 4 MÃ
Rule Import:
1. Khoá chính: Mã Tôm + Loại NL + Mã CĐ + Mã Size – nếu tồn tại → UPDATE
2. Chưa tồn tại → INSERT mới, hệ thống tự sinh Mã LNS
3. Thiếu bất kỳ khoá nào → BỎ QUA dòng đó, ghi log lỗi.
