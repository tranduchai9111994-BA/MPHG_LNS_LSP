BỐI CẢNH DỰ ÁN (đọc trước mọi session)
================================================================================

Tôi là Business Analyst cho dự án HRIS của công ty thủy sản Minh Phú Hậu Giang
(MPHG). Tôi có 1 file prototype HTML duy nhất tên là:

    MPHG_LuongSanPham_Full.html  (~3,683 dòng)

File này là prototype tương tác đầy đủ gồm 2 module chính:
  - LSP: Lương Sản Phẩm (pg1–pg9)
  - LNS: Lương Năng Suất (pg10 với 11 sub-tabs: sys1–4, dm1–2, tl1, fn1–6, fn8)

Cấu trúc kỹ thuật hiện tại:
  - 1 shell HTML chứa sidebar navigation + iframe
  - 10 embedded pages trong thẻ <script type="text/html" id="pgX">
  - Mỗi page là 1 HTML document đầy đủ, load vào iframe qua srcdoc
  - Page pg10 là "app lồng app" với 11 sub-tabs nội bộ
  - Mỗi page có <style> riêng → CSS bị duplicate nặng
  - JS helper functions bị copy paste giữa các page

MỤC TIÊU REFACTOR:
Chuyển từ 1 file monolith → multi-file structure có thể mở rộng,
dùng Live Server (VSCode) hoặc double-click mở browser để chạy.
